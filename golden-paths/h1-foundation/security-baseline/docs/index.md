# Security Baseline

This Golden Path template provides a security-hardened project scaffold following the Three Horizons platform security standards.

## Features

- Pre-configured security policies (OPA/Gatekeeper)
- Container security best practices (non-root, read-only rootfs)
- Secret management via External Secrets Operator  
- Network policies for namespace isolation
- TLS 1.2+ enforcement
- RBAC configuration templates

## Usage

Use this template from the Developer Hub to scaffold a new project with security best practices built in.
