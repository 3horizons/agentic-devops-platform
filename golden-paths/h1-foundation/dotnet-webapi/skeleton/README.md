## Getting Started

### Prerequisites

- .NET SDK 8.0 or later

### Development Setup

```bash
# Restore dependencies
dotnet restore src/

# Run the application
dotnet run --project src/
```

The API will be available at [http://localhost:5000](http://localhost:5000).

Swagger UI is accessible at [http://localhost:5000/swagger](http://localhost:5000/swagger) in development mode.
